Real components, real events.

A React form, a Tiptap editor, a Stripe checkout, a chart from your favorite library — any of it works. Fieldnotes doesn't intermediate your render tree, it just positions it in infinite space.

get started

Six lines to a working canvas.

Core is a single class. Register the tools you want. Everything else is optional.

import {
  Viewport, SelectTool,
  PencilTool, HandTool
} from '@fieldnotes/core';

const viewport = new Viewport(
  document.getElementById('canvas'),
  { background: { pattern: 'dots' } }
);

viewport.toolManager.register(new SelectTool());
viewport.toolManager.register(new PencilTool({ color: '#1a1a1a' }));
viewport.toolManager.register(new HandTool());

// drop a DOM node on the canvas
viewport.addHtmlElement(myCard, { x: 100, y: 200 });
import {
  FieldNotesCanvas, CanvasElement, useCamera
} from '@fieldnotes/react';
import { SelectTool, PencilTool } from '@fieldnotes/core';

function App() {
  return (
    <FieldNotesCanvas
      tools={[new SelectTool(), new PencilTool()]}
      defaultTool="select"
      style={{ width: '100vw', height: '100vh' }}
    >
      <CanvasElement position={{ x: 100, y: 200 }}>
        <TokenCard hp={42} />
      </CanvasElement>
    </FieldNotesCanvas>
  );
}

Bring your own tools.

Tools are a strategy pattern. Want a ruler, a lasso, a hex-aware measuring tape? Register your own — the SDK doesn't care.

  • Store: add, remove, update, event bus
  • Camera: panTo, setZoom, screenToWorld
  • History: undo, redo via command stack
  • Layers: visibility, locking, ordering, opacity
  • Export: PNG Blob with scale & padding
  • Perf: getRenderStats(), logPerformance()

changelog

Every commit, in public.

Ten releases in six weeks. The 0.8.x line was a performance sweep; 0.9 added D&D VTT tools; 0.10 brought rich text formatting.

0.10.0

2026-04-20 added changed fixed
  • Note rich text formatting — inline bold, italic, underline, and strikethrough via contentEditable + execCommand.
  • Floating formatting toolbar — B/I/U/S buttons and font size dropdown. Uses pointerdown + preventDefault to keep focus on the note.
  • Font size presets — Small (14px), Normal (18px), Large (24px), Heading (32px). Configurable via ViewportOptions.fontSizePresets.
  • Headless formatting API — toggleBold(), toggleItalic(), setFontSize(), getActiveFormats() for custom toolbar UI.
  • toolbar: false — disable built-in toolbar and use the formatting API with your own UI.
  • HTML sanitizer — sanitizeNoteHtml() strips disallowed tags/attributes (XSS prevention).
  • Canvas export rich text — renderNoteOnCanvas handles styled runs with word wrapping.
  • Keyboard shortcuts: Ctrl/Cmd+B (bold), Ctrl/Cmd+I (italic), Ctrl/Cmd+U (underline).

0.9.0

2026-04-10 added fixed
  • MeasureTool — drag to measure distances in feet. Snaps to hex/square grid centers. Configurable feetPerCell.
  • TemplateTool — D&D area-of-effect templates. Four shapes: circle, cone, line, square.
  • PHB-accurate hex-filled templates — on hex grids, templates fill actual hex cells.
  • Hex distance — getHexDistance() computes cube-coordinate distance.
  • Template resize via SE handle. Origin marker highlights the origin hex.
  • Image center-snapping to hex/square grid cells on placement.

0.8.x

2026-03-22 → 03-28 performance added fixed
  • PNG export — exportImage() returns a PNG Blob with scale, padding, background & filter options.
  • Per-layer offscreen caching — unchanged layers re-composited without re-rendering.
  • Quadtree spatial index — O(log n) hit-testing & viewport culling.
  • Tiled hex grid rendering — ~250× faster on large grids (4 fps → 1000+ fps).
  • Stroke & arrow geometry caching — computed once at commit time.
  • ImageBitmap pre-decoding — async upgrade to GPU-ready bitmaps.
  • Viewport decomposition: extracted RenderLoop, DomNodeManager, InteractMode.
  • getRenderStats() and logPerformance() instrumentation.

0.5.0 → 0.7.0

2026-03-18 → 03-21 added
  • 0.7.0 — Grid element (square & hex) for D&D maps. HTML element persistence via domId.
  • 0.6.0 — Layer system (visibility, locking, ordering, per-layer opacity), snap-to-grid, note text color.
  • 0.5.0 — Text tool, arrow binding, shapes. React hooks (useSyncExternalStore).

0.1.0 → 0.4.0

2026-03-08 → 03-15 added
  • 0.4.0 — React hooks overhaul. Tool options API.
  • 0.3.0 — Stroke smoothing. Pressure-sensitive stroke width.
  • 0.2.0 — AutoSave (debounced localStorage).
  • 0.1.0 — Initial release. Infinite canvas · pan & zoom · pencil · eraser · notes · arrows · images · undo/redo · HTML embedding.

Origin. Fieldnotes started as the canvas layer for a D&D companion app I was building — players kept using Apple Freeform on an iPad for session notes, and I wanted to give them something that could host the rich card components our app already had.

Every existing option forced a trade-off: Tldraw wanted a commercial license, Excalidraw couldn't embed our components, React Flow wasn't meant for general drawing. So I wrote this instead. It's a pet project — but it's shipping, the core is at zero dependencies, and I'd love for you to poke at it.
